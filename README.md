
# Cowork

A beautiful GUI for [Claude Code](https://docs.anthropic.com/en/docs/claude-code) - your AI-powered development partner.

![Version](https://img.shields.io/badge/version-0.0.4-blue)
![Platform](https://img.shields.io/badge/platform-macOS%20%7C%20Linux-lightgrey)

---

## About

Cowork is a native desktop application that gives Claude Code a polished GUI interface.

**Features:**
- Native desktop UI (macOS & Linux)
- Session management with history
- Real-time streaming output
- Markdown rendering with syntax highlighting
- **Multi-provider support** - [Z.ai](https://z.ai/), [Minimax.io](https://platform.minimax.io/) coding plans, and any OpenAI-compatible API

## From Source

```bash
# Install dependencies

bun run dev

# Build for production
bun run dist:mac    # macOS
bun run dist:linux  # Linux
```
